Easy Problems

1. FizzBuzz: Print numbers from 1 to 100. For multiples of 3, print "Fizz" instead of the number, and for multiples of 5, print "Buzz". For numbers which are multiples of both 3 and 5, print "FizzBuzz".

2. Palindrome Checker: Check if a given string is a palindrome.

3. Factorial Calculator: Write a function that returns the factorial of a given number.

4. List Sum: Write a function that takes a list of numbers and returns the sum.

5. Count Vowels: Write a function that counts the number of vowels in a given string.
